### 正文

最近那个 SBTI 人格测试是不是刷屏了？🌊

我这个周末心血来潮，**用纯静态网页技术复刻了一个平替版**，叫 SBTI+。

花了大概两天，写完直接部署上线，现在谁都能玩了👇

### 🛠️ 技术栈（真的很简单）

- 纯 HTML + CSS + 原生 JS
- **零依赖，零后端，零图片**
- 代码不到 1500 行，打包 < 30KB
- 部署在 Zeabur 静态站，5 秒钟上线
- 统计用 Cloudflare Web Analytics，无 cookie

### 🎨 设计上做了几件事

1. **15 个维度 × 22 个人格**，比 MBTI 细得多
2. **五级稀有度系统**：常见 → 普通 → 稀有 → 史诗 → 传说
3. **传说级带金色呼吸光晕动画**（CSS 纯手写）
4. **头像用 SVG + Emoji 运行时生成**，完全不依赖图片资源
5. 每个人格描述 300~450 字，按**诙谐 / 腹黑 / 温柔 / 毒舌** 等不同基调写

### 🎯 使用效果

测完自己试了一遍：

🕹️ 拿捏者（腹黑版）—— 差点报警，写得太准了
🌷 老母亲（温柔版）—— 直接被治愈
🎲 混沌邪恶（疯狂诙谐）—— 笑到邻居敲墙

### 📚 完整过程写了一篇文章

从"为什么要做"到"技术选型"到"踩的坑"，全在公众号文章里。

**核心心得**：
- 娱乐向产品最贵的资产是"打开即玩"
- 稀有度是传播学上的精确打击
- 描述一定要够长，用户是喜欢被认真解读的

代码全部开源，可以拿去改自己的版本💡
